## ✨ What this app does

Document-Intelligent-Assistant helps turn messy document images into clean JSON data.

It is built for scanned pages, photos of forms, and other document images that are hard to read by hand.

The app uses OCR to read text from images, then uses Claude AI to organize that text into structured output. This makes it easier to search, sort, and reuse the data.

## 🔍 Best input tips

For better OCR results, use documents that are:

- Clear and in focus
- Not tilted
- Well lit
- High contrast
- Free from heavy shadows

If you scan pages yourself, use a straight top-down angle and keep the image size large enough to read small text.
